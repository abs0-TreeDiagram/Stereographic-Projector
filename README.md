# Stereographic-Projector
可将矩形图片进行球坐标逆变换并进行球极投影逆变换操作的脚本，仅供娱乐。

本项目使用了AI辅助编程。

## 如何开始

### 1. 安装/检查Python环境

请自行查阅相关教程。

### 2. 安装所需的库

使用`pip`或其他方式安装所需的库。

具体地，按下`Win+R`，在弹出的运行窗口输入`cmd`，回车，在弹出的命令行窗口键入如下命令：

```
pip install pillow numpy
```

回车，等待安装完成。

### 3. 运行程序

下载并双击运行仓库中的`StereographicProjector.py`，按GUI界面提示操作。

注意，程序中的变换在数学上的像集为整个二维空间，程序输出仅为像集的一部分。

你可以导入仓库中提供的`WorldMap`文件，这是一张**等距圆柱投影**世界地图，经程序处理，你将得到一张**球极投影地图**。
